## Conexion AJAX (Rest)
Para hacer una conexion AJAX, o leer un web service, lo puedo hacer de la siguiente manera
* Hacerlo nativamente con las funcionalidades de javascript (XMLHttpRequest/Fetch)
* Ocupar una libreria que nos automatice el trabajo. Ejemplo Axios.

### instalar axios
En la consola, ejecutar lo siguiente:
> npm install axios

## agregar bootstrap 4
Se puede hacer de dos maneras, uno manual (agregar los estilos a mano), y otra manera es ocupar una libreria.
